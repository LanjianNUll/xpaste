
一个window 的粘贴板应用 

主要功能： 可以记录复制的历史记录，包括图片和文字，可提供选择某个历史记录并粘贴。 

要求： 

多格式支持：除了纯文本和图片，还应支持 HTML (保留格式文本)、文件路径 和 颜色代码。

快速唤起：支持自定义全局热键（如 Alt + V）在鼠标指针处弹出窗口

混合预览：

图片显示缩略图，点击可查看原图。

长文本自动截断，支持搜索关键词高亮。

代码片段自动识别语言并进行语法高亮。

自动分类：系统自动将历史归类为"链接"、"图片"、"文本"、"文件"。

## 新增功能需求

### 1. 点击历史记录直接插入剪贴板
- 点击历史列表中的任意item，直接将内容写入系统剪贴板
- 移除右侧预览面板的"写入剪贴板"按钮
- 保留预览面板用于展示详细内容

### 2. 历史记录按天分类
- 默认显示今天的历史记录
- 支持快速切换：今天、昨天、前天
- 支持自定义日期选择，可查看任意日期的历史记录
- 使用标签页形式展示日期筛选选项

### 3. 图片缩略图显示
- 历史记录列表中，图片类型显示缩略图（60x60px）
- 替代原有的"[图片]"文字显示
- 保持右侧预览面板显示原图
- 提升用户识别复制内容的效率

### 4. 跟随光标的快捷窗口
- 使用全局快捷键 Alt+V 唤起快捷窗口
- 窗口出现在鼠标光标附近位置
- 窗口尺寸：360x500px
- 窗口特性：
  - 始终置顶显示
  - 不在任务栏显示
  - 失去焦点时自动隐藏
  - 点击历史记录后自动关闭
- 窗口内容：
  - 仅包含历史记录列表
  - 顶部包含日期筛选
  - 无预览面板

框架：  tauri + rust  + vue3
模块,推荐技术栈 / 库,实现目标
剪贴板监听,arboard (Rust),跨平台的剪贴板读写库，对 Windows 支持良好。
全局快捷键,tauri-plugin-global-shortcut,处理 Alt + V 等热键唤起窗口。
本地数据库,sqlx + SQLite,存储历史记录，Rust 异步操作保证界面不卡顿。
窗口管理,Win32 API (via windows-rs),实现“失去焦点自动隐藏”和“在鼠标位置弹出”功能。
UI 组件库,Elementplus UI 或 UnoCSS,极简风格，符合 Windows 11 的视觉审美。
